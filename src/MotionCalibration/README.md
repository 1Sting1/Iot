## Управление роботом с динамической настройкой режимов

Этот скетч реализует дистанционное управление двухколесным роботом по Bluetooth с возможностью динамической калибровки. Робот не просто выполняет жестко заданные команды движения, а позволяет пользователю настраивать полярность и скорость каждого из моторов для 4-х основных режимов (Вперед, Назад, Влево, Вправо), используя геймпад.

**Моторы**
- **D4:** Направление 1-го мотора (`DIR1`)
- **D5:** Скорость 1-го мотора (`SPEED1`, PWM/ШИМ)
- **D7:** Направление 2-го мотора (`DIR2`)
- **D6:** Скорость 2-го мотора (`SPEED2`, PWM/ШИМ)

**Bluetooth-модуль:**
- **D12:** RX (подключается к TX модуля)
- **D13:** TX (подключается к RX модуля)

### Структура данных и логика работы

Для хранения настроек используется структура `MovementConfig`, которая хранит состояния направлений (`d1`, `d2`) и значения шим (`p1`, `p2`) для моторов.
Массив `drive[4]` хранит индивидуальные настройки для 4-х режимов движения:
- **0:** Движение вперед (`F`)
- **1:** Движение назад (`B`)
- **2:** Поворот влево (`L`)
- **3:** Поворот вправо (`R`)

По умолчанию при старте системы все режимы инициализируются со скоростью 200 и направлениями `false`.

### Функции управления движением

- `runMotors(mode)`: считывает сохраненные настройки для выбранного режима (направления и скорости) и применяет их к соответствующим пинам драйвера моторов.


- `stopMotors()`: полная остановка двигателей без сброса сохраненных настроек направления.

### Bluetooth команды (Управление и калибровка)

Код принимает символьные команды по Bluetooth. Команды делятся на выбор режима и его настройку:

**Базовое движение:**
- `F` (Forward): Активировать режим "Вперед" (индекс 0) и запустить моторы.
- `B` (Backward): Активировать режим "Назад" (индекс 1) и запустить моторы.
- `L` (Left): Активировать режим "Влево" (индекс 2) и запустить моторы.
- `R` (Right): Активировать режим "Вправо" (индекс 3) и запустить моторы.
- `0`: Остановить моторы.

**Настройка активного режима (кнопки геймпада):**
- `T` (Треугольник): Цикличное переключение комбинаций направления вращения колес (`00`, `01`, `10`, `11`). Позволяет подобрать правильное направление вращения обоих моторов для текущего режима.
- `X` (Крестик): Остановка моторов и фиксация выбранного направления.
- `S` (Квадрат): Плавное увеличение скорости 1-го мотора на 5 единиц (максимум до 255). Полезно для выравнивания движения, если робота уводит в сторону.
- `C` (Кружок): Плавное увеличение скорости 2-го мотора на 5 единиц (максимум до 255).

### Мониторинг

В Serial Monitor на скорости 9600 бод выводится статус системы. При старте появляется сообщение о готовности. При нажатии кнопки `T` (калибровка направлений) в монитор выводится номер текущего настраиваемого режима и номер новой комбинации направлений.
