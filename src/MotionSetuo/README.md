## Управление двухколесным роботом на Arduino

Этот скетч обеспечивает базовое управление роботом с двумя моторами. Реализованы функции для прямолинейного движения, поворотов и разворотов на месте.

### Подключение пинов

- D4: Направление левого мотора (DIR)
- D5: Скорость левого мотора (PWM)
- D7: Направление правого мотора (DIR)
- D6: Скорость правого мотора (PWM)

### Функции управления

Функции принимают аргумент скорости от 0 до 255.

- `forward(speed)`: движение вперед
- `backward(speed)`: движение назад
- `turn_left(steepness)`: плавный поворот налево (вращается только правое колесо)
- `turn_right(steepness)`: плавный поворот направо (вращается только левое колесо)
- `rotate_left(speed)`: разворот на месте влево (колеса вращаются в разные стороны)
- `rotate_right(speed)`: разворот на месте вправо
- `stop_motors()`: полная остановка
