## Прошивка для управления мобильным роботом на Arduino

Данный код предназначен для дистанционного управления колесной платформой через Bluetooth-модуль. В коде реализована логика движения по прямой, реверса, разворотов на месте и остановки.

### Распиновка (Pinout)

**Правый привод:**
- D4: Направление вращения (DIR)
- D5: Регулировка скорости (PWM)

**Левый привод:**
- D7: Направление вращения (DIR)
- D6: Регулировка скорости (PWM)

**Bluetooth (HC-05/06):**
- D10: TX модуля (Arduino RX)
- D11: RX модуля (Arduino TX)

### Основные команды управления

Программа использует следующие функции:

- `goForward()`: движение вперед на заданной крейсерской скорости.
- `goBackward()`: движение назад.
- `spinLeft()` / `spinRight()`: разворот робота на месте (колеса вращаются в противоположных направлениях).
- `curveLeft(factor)` / `curveRight(factor)`: плавный поворот по дуге (параметр `factor` определяет интенсивность).
- `brake()`: немедленное прекращение вращения всех моторов.
